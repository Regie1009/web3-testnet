Apa itu Pharos Testnet?

Jaringan uji coba resmi dari proyek blockchain Pharos sebagai lingkungan pengembangan dan eksperimen sebelum peluncuran mainnet.

Pharos Testnet

  • Token uji coba gratis (dari faucet)
  • Lingkungan pengembangan aman
  • Menguji DApps tanpa risiko finansial
  • Melaporkan bug dan memberikan feedback
  • Simulasi ekonomi token
  • Versi terbaru fitur eksperimental

Pharos Mainnet

  • Token dengan nilai riil
  • Jaringan produksi aktif
  • Transaksi bernilai finansial
  • Ekosistem aplikasi yang berjalan
  • Ekonomi token nyata
  • Fitur stabil dan teruji

Eksplorasi kemampuan unggulan yang ditawarkan oleh Pharos Testnet untuk pengembang dan validator.

Mulai dengan Pharos Testnet

Panduan langkah demi langkah untuk memulai pengembangan dan pengujian di jaringan Pharos Testnet.
1

Create Wallet

Gunakan browser yang support WEB 3 (Brave, Chrome, Mozilla). Install Metamask wallet Extension dan buat akun baru khusus untuk testnet.

Click to Join Testnet

Pastikan Anda tidak menuggunakan Wallet utama Anda. Buat wallet baru khusus Testnet

2

Dapatkan Token Faucet

Kunjungi faucet resmi Pharos Testnet untuk mendapatkan token uji coba gratis:

PHRS Faucet USDT Faucet ZAN Faucet

Masukkan alamat wallet Anda dan dapatkan token uji untuk melakukan transaksi dan menguji aplikasi.

3

Konfigurasi Lingkungan Dev

Instal Pharos SDK dan CLI tools untuk mulai membangun:

npm install -g @pharosnetwork/cli
pharos init my-dapp
cd my-dapp
pharos testnet connect

Eksplor dokumentasi developer untuk panduan lebih lanjut.

4

Deploy & Uji DApp

Deploy kontrak pintar Anda ke jaringan testnet:

pharos contract deploy --network testnet

Gunakan explorer testnet untuk memantau transaksi dan interaksi kontrak:

Buka Testnet Explorer

Pertanyaan Umum

Apa perbedaan utama antara Testnet dan Mainnet?

Testnet adalah lingkungan pengembangan dan pengujian di mana token tidak memiliki nilai riil, sedangkan Mainnet adalah jaringan produksi di mana transaksi melibatkan aset bernilai nyata. Testnet memungkinkan pengujian tanpa risiko finansial.

Apakah token testnet bisa ditukar dengan mata uang nyata?

Tidak, token di Pharos Testnet hanya memiliki nilai uji coba dan tidak dapat ditukar dengan aset nyata atau mata uang kripto lainnya. Token testnet hanya digunakan untuk pengujian fungsionalitas jaringan dan aplikasi.

Bagaimana cara menjadi validator di Pharos Testnet?

Untuk menjadi validator di testnet:

  1. Setup node validator mengikuti panduan teknis
  2. Dapatkan token testnet dari faucet khusus validator
  3. Lakukan bonding token untuk mulai berpartisipasi dalam konsensus
  4. Pantau performa node melalui dashboard validator

Lihat dokumentasi validator untuk petunjuk lengkap.

Apa manfaat berpartisipasi di Pharos Testnet?

Manfaat berpartisipasi:

  • Pengembang: Menguji DApps secara gratis dengan akses fitur terbaru
  • Validator: Pengalaman mengoperasikan node tanpa risiko finansial
  • Pengguna: Mendapatkan reward melalui program bug bounty
  • Semua partisipan: Berkontribusi dalam pengembangan ekosistem Pharos
Apakah ada program insentif untuk partisipan testnet?

Ya, Pharos Testnet menawarkan beberapa program insentif:

  • Bug Bounty Program: Hadiah untuk melaporkan kerentanan keamanan
  • Validator Incentive Program: Reward untuk validator dengan uptime tinggi
  • DApp Competition: Hadiah untuk aplikasi terbaik yang dibangun di testnet
  • Testnet Pioneer NFT: Koleksi digital untuk partisipan awal
