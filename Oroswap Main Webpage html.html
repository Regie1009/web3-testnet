4

Integrasikan DApp Anda dengan OrosSwap Testnet:

// Contoh integrasi OrosSwap SDK
import { OrosSwap } from '@orosswap/sdk';

const swap = new OrosSwap({
  network: 'testnet',
  wallet: connectedWallet

// Lakukan swap token
await swap.swapTokens('tETH', 'tUSDC', '1.0');

Pertanyaan Umum

Bagaimana cara mendapatkan token uji untuk OrosSwap Testnet?

Anda bisa mendapatkan token uji melalui faucet resmi OrosSwap Testnet. Kunjungi halaman faucet, hubungkan wallet Anda, dan pilih token yang Anda butuhkan. Token akan dikirim langsung ke wallet Anda dalam hitungan detik.

Apakah transaksi di testnet memerlukan gas fee?

Ya, tetapi Anda menggunakan token gas uji (tORO) yang tidak memiliki nilai riil. Anda bisa mendapatkan tORO gratis dari faucet untuk melakukan semua transaksi pengujian.

Bisakah saya menjadi liquidity provider di testnet?

Tentu! Testnet adalah tempat sempurna untuk menguji strategi liquidity provision. Anda bisa menambahkan liquidity ke berbagai pool dan mensimulasikan imbalan tanpa risiko finansial.

Bagaimana cara melaporkan bug atau masalah di testnet?

Anda dapat melaporkan bug melalui:

  1. GitHub Issues di repositori resmi
  2. Saluran #bug-report di Discord komunitas
  3. Formulir bug bounty di situs web

Laporan bug yang valid berhak mendapatkan hadiah melalui program bug bounty.

Kapan OrosSwap Mainnet akan diluncurkan?

Mainnet dijadwalkan diluncurkan pada Q4 2023 setelah fase pengujian testnet selesai. Tanggal pasti akan diumumkan melalui saluran resmi setelah audit keamanan lengkap selesai.
